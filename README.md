

### Features

- **Login Screen**: Custom widget creation for the entire app; animations for displaying available categories in the app; custom colors; responsive interface development with flexible widgets; custom text field creation.
- **Registration Screen**: Registration screen creation with a form; field masking; screen scroll when a field is focused.
- **Base Screen**: Pagination of screens; navigation via Bottom Navigation Bar.
- **Home Screen**: Category display with selection; grid product display; search field creation; animation for adding a product to the cart.
- **Product Details Screen**: Display of image, description, price, name; custom quantity widget creation.
- **Cart Screen**: Increment in the custom quantity widget; cart item quantity adjustment and removal; cart item listing.
- **Orders Screen**: Order listing; status display; product listing by order; Pix dialog presentation.
- **Profile Screen**: User data form development; password change dialog; sign-out button.

- **Authentication**: App login; user registration; password recovery; token authentication.
- **Home**: Product listing; product sorting by category; product filtering by name; dynamic product scrolling.
- **Cart**: Product addition; quantity updates; product removal; persistent cart (products remain when exiting the app); checkout process.
- **Order**: Order listing; payment via QR code Pix; payment via Pix copy and paste.
- **Profile**: User data viewing; password update; logout.
